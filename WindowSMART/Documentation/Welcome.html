Welcome to WindowSMART 2015!

Dojo North Software's WindowSMART 2015 Help is designed to provide you a ready means to answer your questions about the WindowSMART software, from how to use it to what various statuses and messages mean. The WindowSMART help is provided in the form of a compiled HTML (CHM) help file, which consists of several tabs to make it easy to find what it is you're looking for.

  • Contents Tab: This tab displays the contents of the help file as books, or chapters. As you expand a book, it can contain one or more chapters (denoted by book icons), and each chapter contains one or more topics (denoted by page icons). The purpose of the chapters, is to organize relevant content together. This makes it easy to navigate from one chapter to the next, and if you're searching by chapter, you can find the desired content in a logical place.
  • Index Tab: This tab displays a list of keywords and phrases. If you're looking for a specific term, you can quickly scan through this topic list. Once you find the topic you want, double-click on it to display the desired page.
  • Search Tab: This tab allows you to search for any term, whether or not the term is listed in the Index tab. All pages in the help file that contain your search term will be displayed, and you can double-click on the desired page to bring it up.

Not sure where to start? Check out the pages in the Overview chapter.
